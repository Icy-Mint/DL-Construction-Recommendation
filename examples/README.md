# Example Building Energy Rules

This directory contains synthetic examples of building energy rules in both text and JSON format.

## Important Note

**NO COPYRIGHTED CONTENT**: All examples in this directory are synthetic and created for demonstration purposes only. They do NOT represent actual ASHRAE 90.1, IECC, or any other copyrighted building energy standards.

## Example Files

- `example_rules_text.txt`: Natural language rule examples
- `example_rules.json`: Structured JSON representation of rules
- `example_building_spec.yaml`: Sample building specification
- `example_output.json`: Sample baseline generation output
